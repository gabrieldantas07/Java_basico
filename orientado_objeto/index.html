Algumas Definições

JavaScript é uma linguagem de programação orientada a objetos e, assim sendo, suas variáveis são dependentes dos objetos. Nas linguagens de programação orientadas a objetos, os programadores criam seus próprios objetos e tipos de variáveis.

As variáveis ​​podem ser usadas para armazenar dados em um programa, como strings, números, objetos JSON ou valores booleanos. Em JavaScript, existem três tipos de variáveis ​​diferentes: var , let , e const . Cada uma dessas variáveis ​​tem várias regras sobre como devem ser usadas e têm características diferentes.

Uma propriedade é um atributo que define uma das características do objeto, como tamanho, cor ou localização na tela, ou um aspecto do comportamento dele, como se o objeto está habilitado ou visível. Para alterar as características de um objeto, deve-se alterar os valores de suas propriedades.
